# Security Policy

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Please do not open a public GitHub issue for security vulnerabilities.**

To report a vulnerability, please [open a private security advisory](https://github.com/OliveMild/main/security/advisories/new) on GitHub.

Include as much of the following information as possible to help us understand and resolve the issue:

- A description of the vulnerability and its potential impact
- Steps to reproduce or proof-of-concept code
- Any suggested mitigations or fixes

We will acknowledge your report and work with you to understand and address the issue promptly.
