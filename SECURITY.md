# Security Policy

## Supported Versions

The following versions of this project are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by opening a [GitHub Security Advisory](https://github.com/OliveMild/main/security/advisories/new).

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- The potential impact of the vulnerability

You can expect an acknowledgement within 48 hours and a resolution or status update within 7 days.

Vulnerabilities that are accepted will be patched and disclosed publicly after a fix is available. Vulnerabilities that are declined will receive an explanation.
