
## Contributing

- Follow [PEP 8](https://peps.python.org/pep-0008/) style (4-space indent, 79-char line limit).
- Include a shebang line (`#!/usr/bin/env python3`) and a `__main__` guard in executable scripts.
- Write tests for any new functionality.
