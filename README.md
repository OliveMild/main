# main

A simple Python "Hello World" project.

## Requirements

- Python 3

## Usage

Run the script directly:

```bash
python hello.py
```

Expected output:

```
Hello World
```

## API Reference

### `hello.py`

The main entry point script. When executed, it prints a greeting message to standard output.

**Entry point:** `__main__` block

| Behavior | Description |
|----------|-------------|
| Output   | Prints `Hello World` to stdout |
| Exit code | `0` (success) |
