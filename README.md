# main

A minimal Python project that provides a simple greeting API.

## Usage

Run the script directly from the command line:

```bash
python hello.py
```

Output:

```
Hello World
```

## API

### `hello.py`

**`greet() -> None`**

Prints `"Hello World"` to standard output.

**Entry point**

When executed as a script (`python hello.py`), calls `greet()` and exits.
