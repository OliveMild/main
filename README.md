# main

A simple Python project demonstrating a Hello World application.

## Overview

This project provides a minimal Python script that outputs a greeting message. It serves as a starting point for building more complex applications.

## Requirements

- Python 3.x

## Installation

Clone the repository:

```bash
git clone https://github.com/OliveMild/main.git
cd main
```

## Usage

Run the script directly:

```bash
python hello.py
```

**Output:**

```
Hello World
```

## API

### `hello.py`

#### `main()`

Entry point of the application. Prints `"Hello World"` to standard output.

**Parameters:** None

**Returns:** None

**Example:**

```python
from hello import main

main()
# Output: Hello World
```
